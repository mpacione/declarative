## What to acknowledge in narration

1. **Project-vocab snapping is firing** — 8 snaps per variant
   means the LLM emitted off-palette values (synth-gen Mode-2
   "purple drift") and the post-IR transform corrected them to
   project-canonical hex / radii / spacing.
2. **The 3-iter cap was conservative** — the demo capped each
   sibling at 3 LLM turns. Bumping `--max-iters` to 8-10 would
   produce richer edits at the cost of API spend.
3. **fontSize snap is currently a no-op** in production — `style.fontSize`
   in IR is binding-resolved string, not numeric. Documented scope
   limit; brand colors / radii / spacing are the dominant levers and
   they all fired.
4. **C4 (variants listing CLI) was deferred** — operator queries
   the variant tree via raw SQL or `dd design log`. Polish, not
   demo blocker.
